# End-to-End Consumer Financial Complaints Analytics Pipeline (CFPB)

---

## 📊 Project Overview
This project involves a comprehensive end-to-end analysis of over **63,000 consumer financial complaints** collected by the Consumer Financial Protection Bureau (CFPB).

The objective was to transform raw, unstructured complaint data into actionable business intelligence using a full-stack data analytics pipeline involving Python, SQL, and Power BI.

This analysis helps identify complaint patterns, operational inefficiencies, seasonal trends, and key areas of consumer dissatisfaction in financial institutions.

---

## 📁 Dataset
Consumer Financial Protection Bureau (CFPB) Complaints Dataset

---

## 🛠️ Tech Stack & Tools
- Data Engineering: Python (Pandas, NumPy)  
- Database Management: MySQL (Joins, Window Functions)  
- Data Visualization: Power BI (DAX, Data Modeling)  
- EDA & Visualization: Matplotlib, Seaborn  
- Process: ETL, Data Cleaning, Feature Engineering, Exploratory Data Analysis  

---

## 🧱 Project Workflow

### 1. Data Cleaning & Preprocessing (Python)
- Handled missing and inconsistent values  
- Standardized categorical variables  
- Converted date fields into proper datetime format  
- Created new analytical features for better insights  
- Mapped 50+ U.S. states into Census Regions for geographic analysis  

---

### 2. Deep-Dive Analysis (MySQL)
- Migrated cleaned dataset into MySQL database  
- Used advanced SQL techniques and window functions (DENSE_RANK)  
- Identified top 3 complaint issues per financial product  
- Performed hierarchical analysis (Product → Sub-product → Issue)  
- Segmented companies based on response efficiency (Timely vs Late)  

---

### 3. Visualization & Reporting (Power BI)
- Designed a 3-page interactive dashboard  
- Built KPIs: Total Complaints, Average Response Time, Timely Response Rate  
- Created regional, product, and company-level insights  
- Used DAX measures for calculated metrics and performance tracking  

---

## 📈 Key Insights Summary

- Seasonal spike in complaints during July (~15%)  
- Checking/Savings & Credit Reporting are major friction areas  
- Smaller companies show higher response delays  
- Regional variation in complaint density is significant
